Mail Messages Easy

Main features

  • View and manage all messages from one source!
    Messages only are shown (system notifications and chat/channel messages are NOT displayed!)
  • Hint: only messages related to records that current user has access to are shown.
    If user can see the message it means that user has at least read access to the record this message is posted to.
    In case you want to hide or show any messages you need to set up access rights/rules to the record.
  • Reply or reply with quote to message thread
  • Forward message to other thread
  • Mark message read
  • Star/Unstar message
  • Configure message deletion rights for users in User Settings
  • Hint: install free Mail Messages Buttons module in case you want to move form buttons to statustbar
  • Use pre-defined filters to display messages you need
  • Open Partner's messages from Partner form view
  • Save messages as draft using free Mail Messages Draft module
  • Show lost messages Click to get Pro Version!
  • Assign message author if author was not detected automatically Click to get Pro Version!
  • Move messages to other thread Click to get Pro Version!
  • Reply , Forward , Move or Delete message directly from Chatter thread Click to get Pro Version!
  • Hide Notifications in Chatter to show messages only Click to get Pro Version!
  • Delete CRM Leads/Opportunities if no messages left except for Notifications Click to get Pro Version!
  • Hint #1: Manage reordering rules easily using free Advanced Auto Reordering Rules Control module
  • Hint #2: Manage multiple email addresses and phone number for Contact using Improved Contacts module

Known limitations / Todo

  • Total number of records in TreeView pager is showing more records than actually exist. This is due to not overload backend with excessive access rules computations.
  • In TreeView last messages from the current page may appear again on the top of the next page. This is because access rules check is done while records are loaded, so messages are pulled from the next page to fill the gap left by messages forbidden by rules.

    • However fixes are on the way!) So feel free to contact me if you have questions or need any support.
