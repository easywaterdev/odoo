3CX Integration

3CX Integration lets you configure your 3CX with an URL to open in Odoo the contact page of the caller (or create it if not found)

Compatible with Odoo SH and On Premise versions


Configuration steps

  • 1/ Install the module
  • 2/ Inside your 3CX configuration, set up the URL to call when receiving a call from a customer to https://your_odoo.com/3cx/%CallerNumber% OR https://your_odoo.com/3cx/%CallerNumber%/%CallerName%
  • 3/ When receiving a call, a web page will open and redirect you to the given customer in Odoo (or create it if not found by phone/mobile)
  • Note: You can check the box "Is 3CX Internal" on a contact to avoid opening it when receiving a call (e.g. your team members)
  • Note: You must be connected to your Odoo (if not you will be redirected to the Login page)
