ZPL and Zebra Printers

Zebra Printing technologies is a market leader in label printing systems. Along with the hardware, the Zebra Programming Language (ZPL) has been developed as a markup language specifically for the Zebra Printers. It ensures that every label you print comes out formatted correctly. The DP ZPL Module allows Odoo to communicate directly with the printer and let’s you one-click print your labels without involving any other software.

Fully customiseable layout and contents

When we say „fully integrated” we mean it. Configure your labels to include any field in Odoo or Odoo-Studio. Labels can be configured on any model in Odoo to cover a wide variety of use-cases (Products, Invoices, Serial numbers). Each label can be assigned to a specific printer, letting you use multiple machines in different locations as well as different label formats for different needs.

Jinja2

Jinja 2 is a web template engine that we used to make our label module even more powerful. How? By letting our users configure loops and conditional statements within their labels, expanding the module capabilities even further. Use loops to automatically print multiple labels for one product, or configure a conditional statement that only prints elements on the label when necessary.

Bulk printing from list view

Need to print out labels for a list of products or entries? Find your list, click print, and the correct label for each product selected will be printed immediately.
