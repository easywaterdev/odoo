Customers/Suppliers Mail/Message History Odoo Apps

This Odoo apps is used to maintain/view Message/email History Of Customers/Suppliers.This Odoo apps Show History of Messages/Emails Sent/Received By Partner. Using this Odoo apps you can easily view the email message history of customer and vendor with send and receive section.

Features

See Total Sent/Receive Mail on Smart Button.

User can see total count of sent/receive mail on smart button.

See List of Sent/Receive Mail.

User can see list of sent/receive mail.

Send Mail Smart Button

In partner form you can see new smart button for Outgoing(Sending) mails.

View Sent Emails
